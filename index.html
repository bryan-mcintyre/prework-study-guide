HTML

  • The body element contains information viewable to the user
  • The section tag helps orgainze your webpage into sections

CSS

  • Margin is the transparent space OUTSIDE an HTML element
  • Padding is the trasparent space INSIDE and HTML element
  • * is a wild card that styels ALL elements on that webpage
